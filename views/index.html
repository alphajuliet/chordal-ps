Chordal API

A musical API focusing on chords and scales. See the README for details.

Example API calls

The paths must begin with /api. These all return JSON.

Chords

  • All chords (/chords)
  • C major (/chord/C/maj)
  • A minor transposed down by 2 semitones (/chord/A/min?transpose=-2)

Scales

  • List all available scales (/scales)
  • A mixolydian mode transposed up by 4 semitones (/scale/A/mixolydian?transpose=4)

Notes

  • All notes in the octave (/notes)
  • Transpose C, D#, and E up by 7 semitones (/note/C,D%23,E?transpose=7)
